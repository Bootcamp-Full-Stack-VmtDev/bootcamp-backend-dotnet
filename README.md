## 📑 Descripción

Este repositorio contiene todos los ejercicios, prácticas y proyectos del módulo **.NET Backend** del Bootcamp **Full Stack Developer**.

Incluye desde fundamentos del lenguaje **C#**, hasta la construcción de **APIs REST** aplicando principios de **arquitectura limpia** y buenas prácticas profesionales.

## 📂 Estructura del repositorio

```bash
📁 1. evolucion-y-anatomia-dot-net
    📁 CsharpIntroduction
    📁 PracticeCalculator

📁 2. guia-definitiva-de-arquitectura
    📁 PracticeUdemy
    📁 TalentInsights

📁 laboratorio
    📁 LaboratorioUdemy
    📄 README.md
```

## 🏷 Módulos y contenido

1. Evolución y anatomía de .NET

🧠 Introducción al ecosistema .NET y fundamentos de C#:

* `CsharpIntroduction` – Conceptos básicos del lenguaje
* `PracticeCalculator` – Aplicación práctica con lógica básica

2. Guía definitiva de arquitectura

🏗 Implementación de buenas prácticas en aplicaciones reales:

* `PracticeUdemy` – Introducción a APIs y estructura por capas
* `TalentInsights` – Proyecto con separación de responsabilidades, DTOs y servicios

3. Laboratorio – API completa

🚀 Proyecto práctico integrador con arquitectura limpia:

* CRUD de instructores
* Manejo de requests y responses
* Separación en capas (Application, WebApi, Shared)
* Persistencia en memoria

📌 Para más detalles revisar: [`Laboratorio`](./laboratorio)
